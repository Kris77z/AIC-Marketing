# AIC-Marketing

这个仓库现在承载的是 `AIC-Marketing` 的实现与参考资料。

## 目录

- `banner-web/`
  - Next.js 应用
  - 统一出稿、自动热点、图库、素材库、场景管理
- `banner-image-design_副本/`
  - 早期 skill / 参考文档 / Python 脚本
- `PHASE-1.5-OVERVIEW.md`
  - 当前 Phase 设计说明
- `PROGRESS.md`
  - 过程记录

## 当前产品方向

- 统一出稿：AI 生图和 HTML 设计收敛到同一入口
- 自动热点：从热点 feed 自动采集并触发出稿
- 图库追踪：所有产出都能回到图库复用

## 开发入口

```bash
cd banner-web
npm install
npm run dev
```

详细说明见 [banner-web/README.md](./banner-web/README.md)。
